<Tabs>
  <TabItem default label="Automatic Installation" value="auto" >
    You can install the addon and add it to your active addons automatically using storybook's [`storybook add`](https://storybook.js.org/docs/api/cli-options#add) option. This will install the addon in a single step:

    ```bash npm2yarn
      npx storybook add storybook-addon-badges
    ```

  </TabItem>
  <TabItem label="Manual Installation" value="manual" >
        
    If you'd prefer to install the addon manually, it's a quick, two-step, process:

    ### Install the addon

    ```bash npm2yarn
      npm i -D storybook-addon-badges
    ```

    ### Add to your storybook config

    ```js title=".storybook/main.ts"
    // Replace [your-framework] with the framework you are using (e.g., react-webpack5, vue3-vite)
    import type { StorybookConfig } from '@storybook/[your-framework]';

    const config: StorybookConfig = {
      // ...rest of config
      addons: [
        '@storybook/addon-essentials'
        'storybook-addon-badges', // ðŸ‘ˆ Register the addon here
      ],
    };

    export default config;
    ```

  </TabItem>
</Tabs>
